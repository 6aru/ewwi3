## ⚡ Overview

This repository provides a distraction-free, feature-rich **i3** configuration tailored with an **E-Ink inspired dark theme**.[cite: 2] Everything is pre-configured to work seamlessly, blending minimalism with high functionality.[cite: 2]

### Key Features
* **Aesthetic**: Minimalist E-Ink dark theme.[cite: 2]
* **Window Management**: Clean i3wm setup with Autotiling enabled.[cite: 2]
* **Widgets (Eww)**: Custom status bar, control center, dashboard, weather, and notifications.[cite: 2]
* **Applets (Rofi)**: Fast app launcher and power menu.[cite: 2]
* **Compositor**: Picom integration for smooth rendering.[cite: 2]
* **Media & System**: Volume and brightness popups, MPD support, and browser media controls.[cite: 2]
* **Lock Screen**: Secured beautifully with i3lock-color.[cite: 2]
