# Clean, Minimal & Feature-Rich i3 + Eww Setup

A lightweight and modern i3 window manager setup for **Debian 13**, featuring a clean E-Ink inspired dark theme, Eww widgets, Rofi, Picom, and carefully chosen defaults for a distraction-free desktop.

## Features

- Clean E-Ink inspired dark theme
- Minimal, distraction-free desktop
- Eww status bar and widgets
- Rofi application launcher & power menu
- Picom compositor with animations
- i3lock-color lock screen
- Autotiling support
- Media controls
- Volume & brightness popups
- Notification center
- Workspace indicators
- Lightweight and optimized for Debian 13
