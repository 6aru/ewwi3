# 🌑 Clean, Minimal & Feature-Rich i3 + Eww Setup

A lightweight and modern **i3** desktop for **Debian 13** with an **E-Ink inspired dark theme**, **Eww widgets**, **Rofi**, **Picom**, and carefully selected defaults for a clean, distraction-free experience.

## ✨ Features

- 🌑 E-Ink inspired dark theme
- 🪟 Clean i3 window manager
- 📊 Eww status bar & widgets
- 🚀 Rofi launcher & power menu
- 🎨 Picom compositor
- 🔒 i3lock-color lock screen
- 📐 Autotiling
- 🎵 Media controls
- 🔊 Volume & brightness popups
- 🔔 Notification center
- 🖥️ Workspace indicators
- ⚡ Lightweight & fast
- ❤️ Built specifically for Debian 13
